routerLink example

All the required and optional query parameters are Observables. First time this component gets initialized the message GithubProfileComponent -> ngOnInit gets displayed in the console. If we press the buton bellow we will navigate to the same component but this time the ngOnInit() will not be called but the parameter will be updated with the new value 1234.

The code:
<button class="btn success" [routerLink]="['/section10/followers', 'nextfollower', '1234']"> Next follower </button>

Router.navigate example

If we press the button below we will navigate programatically to the followers page passing the optional query parameters: fromId and fromUser. Check the url after you have clicked.

The code:
<button class="btn success" (click)="back()">Back to List</button>
The code:
...
constructor(
  private route: ActivatedRoute,
  private router: Router
) { }
...
back() {
  this.router.navigate(['/section10/followers'], {
  queryParams: {
   fromId: this.followerId,
   fromUser: this.user
  }
  });
}
