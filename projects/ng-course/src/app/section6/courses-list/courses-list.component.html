In this example I demo the use of the angular templates and the *ngIf directive. Click the + and - to increment the number of courses. A specific template that shows the message "No courses yet" is used in case we don't have any course. If we have courses a template that displays the message "Number of courses is ..." is used.


Number of courses is {{courses.length}} No courses yet
The code:
<div *ngIf="courses.length > 0; then coursesList else noCourses"></div>
<ng-template #coursesList>
  Number of courses is {{courses.length}}
</ng-template>
<ng-template #noCourses>
  No courses yet
</ng-template>
