Example of ngFor directive in conjunction with trackBy Open the javascript console and inspect the button. Click the load button and noticed that the trees are loaded only once no matter how many times you press the button. You can tell this because the <li> (list item) elements get purple bliking only once. This is very handy when we manipulate a lot of data. This is because I use *ngFor="let tree of trees; trackBy trackData" where trackData is a function defined in my typescript class with the following implementation:

trackData(index, tree) {
 return tree ? tree.id : undefined;
}
